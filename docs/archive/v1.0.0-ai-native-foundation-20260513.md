## Release position

This is the first stable v1.0 foundation of Visualized Deadline.

VD is no longer only a visual deadline matrix or a generic task manager. This version establishes the foundation of an AI-native cognitive task structuring system.

Core philosophy:

> Users should not merely manage tasks.
> Users should advance their life.

## Core systems included

### 1. Visual pressure system

- Deadline pressure matrix
- Homepage urgent deadline hotspot
- Time-pressure visualization
- Task display progress and pressure cues

### 2. AI task cognition

- Local user-provided AI API configuration
- OpenAI-compatible and DeepSeek-compatible request flow
- AI task analysis panel
- Structured cognitive report generation
- AI task intake command bar
- Natural language to task draft workflow
- User confirmation before task creation

### 3. Dual progress foundation

- Manual task progress
- Time-based display progress
- Progress mode handling
- Future-ready separation between actual task progress and time consumption

### 4. Task page foundation

- Dedicated Task module
- Full matrix and task list
- AI analysis and intake surfaces
- Task create/edit/archive/delete flows

### 5. Life structure foundation

- Life Tree direction
- Strategic module layout
- Foundation for future roadmap and goal systems

### 6. Social graph foundation

- Social graph visualization
- Favorability-based distance model
- Contact list management and sorting
- Relationship editing flow

### 7. Review foundation

- Review page direction
- AI review prompt/service foundation
- Future weekly/monthly cognitive review support

### 8. Local-first developer foundation

- LocalStorage persistence
- Developer tools panel
- Data safety panel
- AI settings stored locally
- No backend dependency yet

## Technical notes

- React + TypeScript + Vite
- Frontend-first architecture
- LocalStorage persistence
- AI service layer under `src/services`
- Modular AI prompt builders
- No heavy backend
- No analytics yet
- User API key stays local to the browser

## Why this version matters

This version proves the core loop:

1. User creates or imports tasks.
2. VD visualizes deadline pressure.
3. AI reads real task data.
4. AI returns cognitive analysis.
5. AI can convert natural language into task drafts.
6. User confirms before system mutation.

This is the first working AI-native VD foundation.
